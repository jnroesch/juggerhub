The base, applied four ways

Centered wordmark header and structured footer are now the shared base — identical across all four. Only the body block changes: verification, team invite, password reset, and team news. Notice the footer's "why you're getting this" line is tailored per email, and content leans on JuggerHub patterns (team card, avatars, mono for times & counts). Fields in braces like {team_name} are merge tags.

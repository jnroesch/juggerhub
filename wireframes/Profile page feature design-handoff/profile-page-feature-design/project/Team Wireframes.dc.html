3Team space — mobile · admin view · create · invite · accept/decline · roles
◆Mobile only · admin (owner) view — every screen shows the full-control version. Member & public views come later.
◆Team type — a city team (has a home city) or a Mixteam (players from different cities who don’t train together but enter tournaments as one crew — no city).
◆Team page = tabs: Members · Activity · News · Trainings. Trainings screens come in the next turn — the tab is shown greyed (Trainingsnext).
◆Roles: admin ↔ member · at least one admin must remain · only admins can delete the team.
◆Invite: sharing a link leads (expires in 7 days); searching users is the backup. Names/dates are placeholder.
